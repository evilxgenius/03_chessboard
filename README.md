групуємо стилі. Дублювання коду ось тут:

```css
.line:nth-child(2n) .block:nth-child(2n){ 
    background: lime;
}
```

```css
.line:nth-child(2n-1) .block:nth-child(2n-1){ 
    background: lime;
}
```

вирішується таким чином:

```css
.line:nth-child(2n) .block:nth-child(2n),
.line:nth-child(2n-1) .block:nth-child(2n-1) {
    background: lime;
}
```
